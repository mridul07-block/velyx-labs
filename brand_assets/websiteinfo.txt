Build a fully responsive multi-page website for “Velyx Labs”, an AI automation and strategy agency that helps startup founders scale their businesses.

The site should be built using:

React.js
Tailwind CSS
React Router DOM for page navigation
Framer Motion for animations
No backend required
Component-based architecture
Clean, production-ready, modern code

The visual direction should be a dark premium futuristic aesthetic, inspired by high-end Web3/DeFi websites, but adapted for an AI agency. The UI must feel atmospheric, polished, elegant, and highly animated.

Overall Visual Style

Match this style across the entire website:

Deep dark background: #0d0d18 and #0a0a0a
Strong violet accent color: #7F77DD
Light violet hover tone: #9B94E8
White typography for headlines
Mid-gray text for supporting content
Glassmorphism cards with subtle blur
Large glowing violet orb / planet-like background element in hero sections
Soft particle / star field in the background
Premium, minimal, futuristic layout
Smooth transitions and subtle motion everywhere
Noise texture overlay at very low opacity for depth

Use Tailwind CSS custom theme tokens for all colors, shadows, spacing, and blur styles.
